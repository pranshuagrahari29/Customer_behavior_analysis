# 🛒 Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using transactional
data of 3,900 purchases. It uncovers insights into spending patterns,
customer segments, and product preferences.

## 📊 Dataset Information

-   Records: 3,900
-   Features: 18
-   Includes demographics, purchase details, and behavioral data

## 🧹 Data Preprocessing

-   Missing value handling
-   Feature engineering
-   Data cleaning using Pandas

## 🗄️ SQL Analysis

-   Revenue by gender
-   Customer segmentation
-   Top products analysis
-   Subscription insights

## 📈 Dashboard

Built using Power BI: - KPIs (Customers, Avg Spend, Ratings) - Category
& age analysis

## 💡 Business Recommendations

-   Improve subscriptions
-   Loyalty programs
-   Optimize discounts
-   Targeted marketing

## 🛠️ Tech Stack

-   Python
-   PostgreSQL
-   Power BI

## 🚀 How to Run

1.  Clone repo
2.  Install dependencies
3.  Run scripts
4.  Open dashboard

## 📌 Conclusion

End-to-end data analysis project using Python, SQL, and BI tools.
